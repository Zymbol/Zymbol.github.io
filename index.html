NOTTRON

Nottron is an application made for Android. It is an original Tron clone my Software Engineering group made for our final project. My main contribution involved the user interface for the menu. I leveraged the power of XML and MotionLayout to make this a visually satisfying entry to the game.


NotCarGurus

NotCarGurus was the first project I was able to really stretch my legs and get a feel for SQL and PHP. I was able to put all my effort into this Database Systems final project. I used HTML, CSS, PHP, mySQL, JavaScript, and a package called AOS for the motion animations. Customers are able to sign up, sign in, login, logout, and view all the vehicles in the database. Employees are able to see where the vehicles are, who ordered them, and where they are being delivered. Managers are able to look at other employee's information and access warehouse information.

RunnerChat

RunnerChat was a great attempt at making a pure PHP/SQL chat application. It was a huge success with the short time we where alotted. Users can log in and message eachother on this website. It will autoscroll to the bottom of the chat and update the database everytime the user types a message. If you are logged in, your messages will appear on the right in blue, if you are receiving a message, they will appear on the left in yellow. Sign up by creating a username and password, send a message, and see it update in real time.
